The New Voyage Planner Modules
We are delighted to announce the launch of OneOcean 2021. This much anticipated release to the OneOcean platform has seen over 1100 improvements and new unctionality built in.
OneOcean 2021 focuses on improving digital workfl ows, the scale of our databases and single view capability to provide a more intuitive, customisable approach to Voyage Planning.
Read a summary of the changes here

The Updates

  • Voyage Planner combines Routes & Passages with Passage Planning to design a single view module to create routes and build passage plans.
  • Customise voyage parameters to inform alerts, event waypoints and improve optimisation
  • Digital passage plans are built-in to enable dynamic updates and collation of data to create plans quickly and accurately.
  • Overlay layers of data such as environmental, weather and ENCs are viewable directly in the module - streamlining the workfl ow in one view

Other Updates

Weather

Weather has been completely reworked to display using the NOAA scale, There is now a built-in simulator that uses forecasts to show how weather systems will affect vessels on a route.

Ports

The Ports database has been ex- panded to include details about facilities and documentation. Preferred Ports can be set allow- ing data updates to be more ef- fectively targeted.

Environmental

EnviroManager now includes regulations from over 170 na- tions. Confi guration options enable the display of relevant regulations to vessels and equipment, with built-in source data for transparency.
