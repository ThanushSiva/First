HTML ELEMENTS : The declaration defines that this document is an HTML5 document The element is the root element of an HTML page The element contains meta information about the HTML page The element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab) The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc. The <h1> element defines a large heading The <p> element defines a paragraph HTML BASICS : All HTML documents must start with a document type declaration: <!DOCTYPE html>. The HTML document itself begins with <html> and ends with </html>. The visible part of the HTML document is between <body> and </body>. DOCTYPE DECLARATION : The <!DOCTYPE> declaration represents the document type, and helps browsers to display web pages correctly. It must only appear once, at the top of the page (before any HTML tags). The <!DOCTYPE> declaration is not case sensitive. HEADINGS : HTML headings are defined with the <h1> to <h6> tags. <h1> defines the most important heading</h1>. <h6> defines the least important heading</h6> PARAGRAPH : HTML paragraphs are defined with the <p> tag </p> LINKS : HTML links are defined with the <a> tag </a> EG : <a href="https://www.w3schools.com">This is a link</a> IMAGES : HTML images are defined with the <img> tag EG : <img src="w3schools.jpg" alt="W3Schools.com" width="104" height="142">
